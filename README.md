# 🎨 Etch-A-Sketch

A browser-based **Etch-A-Sketch Drawing App** built with HTML, CSS, and JavaScript. Users can draw on a dynamic grid by simply hovering over cells, creating pixel-art style sketches directly in the browser.

## 🚀 Features

* Interactive drawing grid
* Hover-to-draw functionality
* Adjustable grid size
* Clear/Reset board option
* Responsive design
* Smooth and intuitive user experience
* Lightweight and fast performance
* 
## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

```bash
etch-a-sketch/

```

## 🎯 How It Works

1. Open the application in your browser.
2. Move your mouse over the grid to draw.
3. Create unique pixel-art designs.
4. Use the reset button to clear the canvas.
5. Adjust grid size (if available) for different drawing experiences.

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/saurabhsingh72487-hub/etch-a-sketch.git
```

2. Navigate to the project directory:

```bash
cd etch-a-sketch
```

3. Open `index.html` in your browser.
